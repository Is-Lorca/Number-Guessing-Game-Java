## â–¶ï¸ How It Works

1. The program displays a welcome message and explains the game rules.
2. The user chooses a difficulty level:
   - Easy
   - Medium
   - Hard
3. Based on the chosen level:
   - A random number is generated.
   - The user receives a limited number of attempts.
4. The user tries to guess the number.
5. After each incorrect guess, a hint is shown:
   - The number is higher or lower.
6. The game ends when:
   - The user guesses correctly, or
   - The attempts run out.
7. The user can choose to play again or exit.
